MFDecoupling.setup_calculation — Method
setup_calculation(fp1::String, fp2::String, L::Int; mode::Symbol=:real)
setup_calculation(X0::Vector, L::Int; mode=:real)

Sets up the calculation. Returns start vector X0 and function rhs for the right hand side of differential equation. Mode can either be :real or :complex. The former uses a real vector representation (with twice as many vector entries) and is the (much fast) default.

It may be useful for performance, to read inputs only a single time with read_inputs, which returns X0.
